Game Development: Cubey

-----------------------------------

Cubey was my first game that I developed in around 3 days. At the time I had played around with the idea of making a game in 2D in the same game engine but i never really got anywhere with it. I decided to pick the Unity game engine as my way of developing the game as it uses real code in the C# coding language and it's beginner friendly for someone like me. a few images of that same game. Cubey.

Extra information about Cubey

-----------------------------------

Cubey originally started as a 2D platformer but I scrapped the idea since I thought it would be too hard at the time.

Cubey was planned to have over 20 levels but i scrapped the idea as I thought I would get burned out too quickly

I made Cubey in an earlier version of the Unity game engine just because I thought it looked nice only to realise it doesnt have as many features as the later versions.
